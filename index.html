Mineplex-Gadgets

Mineplex Gadgets All of Mineplex's Gadgets, in a chest... This maven thingy was made by @xiurobert... You MAY fork, you MAY make a pull request...

Compiling

  1. Download the .zip
  2. Install maven
  3. Extract the contents of the zip somewhere
  4. Get its location and in the terminal (Mac + linux) $cd /where/you/extracted/the/contents or command prompt (Windows) C:\> cd "C:\where\you\extracted\the\file".
  5. Then mvn clean package or mvn clean install(DO ONLY ONE!!!)
